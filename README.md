# 🎓 Student Management App (Lite)

A lightweight **Student Management System** built with **React** to practice modern frontend development concepts such as Routing, Forms, Validation, CRUD operations, and Local Storage.

> **Current Version:** Frontend Only (React)
>
> **Future Version:** Full MERN Stack (MongoDB + Express + React + Node.js)

---

## 🎯 Project Goal

This project is designed as a learning project to master:

- React Fundamentals
- Component Architecture
- React Router
- Forms & Validation
- CRUD Operations
- Local Storage
- Clean Project Structure

The backend is intentionally excluded from the current version so the focus remains entirely on frontend development.

---

## 🛠️ Tech Stack

### Frontend

- React
